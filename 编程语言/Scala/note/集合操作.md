# set

不重复无序集合，定义可变set需要到scala.collection.mutable下的
```
import scala.collection.mutable._

val s = HashSet(1,2)
val s1 = HashSet(3,4)
s.add( 3 )
s ++ s1
s ++= s1
```


# map

默认不可变
```
import scala.collection.mutable._
val mp = Map[String, Int]("a" -> 1)

mp("a") = 2
mp.put("b" -> 2)

mp += "b" -> 2
mp += (("c", 3))

mp.remove("a")
mp.-=("a")

mp.get("a")  返回为对象
mp.get("a").get  获取值

```
