# 会计学原理

## 复式记账

1. 特点

    **有借必有贷，借贷必相等**

## 会计恒等式

1. 反应资金在一个时点上的使用情况及其来源

    > 资产 = 负债 + 所有者权益
    
    **资产** (借方)： 资金的使用结果  
    **负债、所有者权益** （贷方）: 资金的来源

2. 反应资金在一段时间内的运动情况

    > 收入 - 费用 = 利润
