# 自动生成requirements.txt

    pip install pipreqs
    pip3 install pipreqs --timeout 1000 -i http://pypi.douban.com/simple/  --trusted-host pypi.douban.com
   
 

# 使用

   pipreqs --debug  --encoding utf8 --force ML-face-recognition/

# 安装

    pip install [-r] requirements.txt
